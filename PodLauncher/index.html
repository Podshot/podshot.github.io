Welcome to PodLauncher



PodLauncher is an Open Source Minecraft Launcher written in Java. PodLauncher is designed to be reliable, light-weight and to be extensible.
PodLauncher comprised of many open source projects, including JSON.Simple, Tomsik68's MCLauncher API, and JSON.Smart
PodLauncher's goal is to maintain an Open Source Minecraft Launcher project and allow anyone to contribute, from ideas and artwork to code and tutorials. PodLauncher consists of two parts, the Launcher, and the Updater.
Both of which can be found on there respective Github Repositories: PodLauncher Repository and Updater Repo.

To download the latest stable PodLauncher build, please click this link: Download PodLauncher or you can run Podlauncher from it's source located on its corresponding Repository.
Details on how to setup the source can be found on the "Installing from Source" section in the README.md
